================================================
   BEFORE YOU START (IMPORTANT!)
================================================

1. INSTALL NODE.JS
   Download: https://nodejs.org/
   Get the LTS version (recommended)

2. INSTALL MONGODB (Choose one):

   Option A - Local:
   → https://www.mongodb.com/try/download/community
   → Install and run: mongod

   Option B - Cloud (Easier):
   → https://www.mongodb.com/cloud/atlas
   → Create free account
   → Get connection string
   → Update in backend\.env

3. GET CLAUDE API KEY (Optional but recommended):
   → https://console.anthropic.com/
   → Sign up (free $5 credits)
   → Create API key
   → Update in backend\.env

================================================
   IF YOU GET ERRORS
================================================

ERROR: "'Doctor' is not recognized"
   → This is caused by spaces in folder path
   → SOLUTION: Use the batch files provided
   → They bypass this issue automatically!

ERROR: "Cannot find module vite"
   → Run: SETUP-AND-RUN.bat
   → It will install everything

ERROR: "MongoDB connection failed"
   → Make sure MongoDB is running
   → Or use MongoDB Atlas (cloud)

ERROR: "Port already in use"
   → Close other instances
   → Or use: CHECK-HEALTH.bat

================================================
   STEP-BY-STEP MANUAL START
================================================

1. Open Command Prompt as Administrator

2. Install Backend:
   cd "E:\Sri\Smart Health & Doctor Appointment System\healthcare-plus\backend"
   npm install

3. Install Frontend:
   cd "E:\Sri\Smart Health & Doctor Appointment System\healthcare-plus\frontend"
   npm install

4. Configure Backend:
   Edit: backend\.env
   Add your MongoDB URI and Claude API key

5. Start Backend (Terminal 1):
   cd backend
   node server.js

6. Start Frontend (Terminal 2):
   cd frontend
   node node_modules\vite\bin\vite.js

7. Open Browser:
   http://localhost:3000

================================================
   FIRST TIME USING THE APP
================================================

1. Application opens at: http://localhost:3000

2. Click "Sign up" to register

3. Choose role: Patient

4. Fill in your details

5. Login with your credentials

6. Add a health record to see AI advice!

7. Book an appointment with a doctor

================================================
   CREATING DOCTOR ACCOUNTS
================================================

To book appointments, you need doctors:

1. Logout from current account

2. Register new account with role: Doctor

3. Login as doctor

4. Complete doctor profile

5. Now patients can book appointments!
